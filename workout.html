Meal Plan Workout

MWF Upper + Core/Abs Plan (No Knee Loading)

Focus: more defined arms and faster pull-up/dip progress. Knee-loading/bending is removed entirely. Format: Push → Pull supersets + Arm superset + Core/Abs block (no floor transitions).

Mon / Wed / Fri No knee loading/bending Anchors: Pull-ups • Dips (every session) Row rotates M/W/F Arms: 3 sets each day

How the supersets work

A superset is two exercises done back-to-back before you rest. It does not add extra exercises. Rest 45–75 sec after each Push→Pull pair (60–90 sec on heavier sets).

“3 sets each” example:
Set 1: Push (set 1) → Pull (set 1) → Rest 45–75 sec
Set 2: Push (set 2) → Pull (set 2) → Rest 45–75 sec
Set 3: Push (set 3) → Pull (set 3) → Rest 45–75 sec

Fast pull-up/dip progress rules

Monday (strength): heavier assistance, lower reps.
Wednesday (volume): lighter assistance, more reps.
Friday (quality): moderate assistance + add a brief hold or slow lower.
Micro-progression: reduce assistance one notch OR add 1 rep total each week.
